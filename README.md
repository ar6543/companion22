# Лонгрид «Компаньон»

Лендинг-лонгрид о умной колонке с ИИ для заботы о пожилых родителях на расстоянии.

## Состав

```
index.html        — вся страница, без зависимостей (шрифты грузятся с Google Fonts)
images/
  robot.png            — герой (робот-колонка)
  companion-home.png   — блок «Как это работает»
  story-nadezhda.jpg   — история 1 (Надежда)
  story-lyubov.jpg     — история 2 (Любовь)
  story-ivan.jpg       — история 3 (Иван Петрович)
```

## Как открыть локально

Просто откройте `index.html` в браузере — или запустите локальный сервер:

```bash
python3 -m http.server 8000
# затем откройте http://localhost:8000
```

## Публикация на GitHub Pages

1. Создайте репозиторий и загрузите туда `index.html` и папку `images/` (структуру сохраните).
2. **Settings → Pages → Build and deployment → Source: Deploy from a branch**, ветка `main`, папка `/ (root)`.
3. Через минуту сайт будет доступен по адресу `https://<ваш-логин>.github.io/<репозиторий>/`.

## Правки

- Текст, цвета, фото меняются прямо в `index.html`.
- Чтобы заменить фотографию — положите новый файл в `images/` и поправьте `src` у соответствующего `<img>`.
